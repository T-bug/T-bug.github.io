css定位

1. static(标准流)

流动模型是默认的网页布局模式。也就是说网页在默认状态下的 HTML 网页元素都是根据流动模型来分布网页内容的。

两个特征：

行内元素在同一行内从左到右横向排列。
块级元素占满整个一行，在页面中自上而下垂直排列，默认状态下，块状元素的宽度为100％ 。

2. relative(相对定位)

以标准流的排版方式为基础，然后使盒子相对于它在原本的标准位置偏移指定的距离。依据left，right，top，bottom等属性在正常文档流中偏移自身位置；使用相对定位的盒子仍然在标准流中，且偏移前的位置保持不动。

 div{
     width : 200px;
     height: 200px;
}

 .b{

    background-color: red;
}

 .a{
    position: relative;
     top:200px;
     left:200px;
     background-color: blue;

    }

3. absolute(绝对定位)

absolute：绝对定位，将元素从文档流中移出来，然后使用left、right、top、bottom属性相对于其最接近的一个具有定位属性的父元素进行绝对定位。如果不存在这样的元素，则相对于body元素，即相对于浏览器窗口，使用绝对定位的盒子脱离了文档流，原先的位置占位关闭。

.a{
    position: absolute;
     top:300px;
     left:300px;
     background-color: blue;

    }

4. fixed(固定定位)

固定定位，类似于决定定位，但是以浏览器窗口为基准进行定位，不会随着滚动条的滚动而改变位置。
